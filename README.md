# Arima Bedside Care (U) Ltd - Website

A professional website for Arima Bedside Care, a healthcare provider in Uganda offering compassionate bedside care services.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional interface with brand colors (blue, green, red)
- **Service Showcase**: Detailed information about all available services
- **Contact Section**: Multiple ways to get in touch (phone, WhatsApp, email)
- **Contact Form**: Direct messaging capability
- **Smooth Navigation**: Easy-to-use navigation bar with smooth scrolling

## Services Offered

1. **Bedside Nursing Care** - Professional nursing services at home
2. **Elderly & Home Care** - Specialized care for seniors
3. **Post-Surgery & Recovery Care** - Post-operative support
4. **Palliative & Chronic Illness Care** - Compassionate care for chronic conditions
5. **Companionship & Daily Living Support** - Emotional support and daily assistance

## How to Use

1. Clone the repository
2. Open `index.html` in your web browser
3. Customize content as needed

## Customization

### Colors
The website uses CSS custom properties (variables) defined in `styles.css`:
- `--primary-blue`: #003da5
- `--secondary-green`: #28a745
- `--accent-red`: #dc3545

Modify these values to change the color scheme.

### Content
Edit the relevant sections in `index.html` to update:
- Service descriptions
- About text
- Contact information
- Company information

### Contact Form
The contact form currently shows an alert. To make it functional:
1. Set up a backend service to receive form data
2. Update the `handleSubmit()` function in `script.js` to send data to your server
